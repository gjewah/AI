When you search for a record, usually Odoo shows matching records by name.
But sometimes there are multiple records with the same name,
and you may want to identify a record by other fields.
With this module, you can customize the display of records, per model.
And you can search on the fields that are used in the display name.

Example for tasks: "P{project_id.id:>03}/T{id:>04} - {name}" ---> "P001/T0001 - My Task"
