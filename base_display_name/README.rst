When you search for a record, usually Odoo shows matching records by
name. But sometimes there are multiple records with the same name, and
you may want to identify a record by other fields. With this module, you
can customize the display of records, per model. And you can search on
the fields that are used in the display name.

Example for tasks: "P{project_id.id:>03}/T{id:>04} - {name}" --->
"P001/T0001 - My Task"

Configuration
=============

To set a custom display_name pattern:

1. Go to *Settings > General Settings > Permissions*.
2. Activate *Custom Display Name* and save.
3. Follow the link below *Custom Display Name* or go to *Settings >
   Technical > Database Structure > Models*.
4. Click on a model to open it.
5. Set a custom *Display Name* pattern and save.

Usage
=====

The display_name of a record is used in:

- The search box
- Many2one dropdown list
- The ``display_name`` field
