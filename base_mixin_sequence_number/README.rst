All records get an id. But you may want to number records differently.
Odoo sequences provide flexible numbering. With this module, you can
number records with a sequence per model.

A record without a name will get the sequence code as the name.

To show the sequence code in display_name, use the module
``display_name``.

Configuration
=============

A record will get a sequence code when a field for this is specified on
the model.

1. Go to *Settings > General Settings > Permissions*.
2. Activate *Use Custom Sequences* and save.
3. Follow the link to *Settings > Technical > Database Structure >
   Models*.
4. Click on a model to open it.
5. Decide which field will store the sequence code. Make a new field if
   necessary.
6. *Store sequence in* - Set the field to store the sequence code.
7. Save. A sequence will be created.
8. Go to *Settings > Technical > Sequences & Identifiers > Sequences*.
9. Customize the sequence.

Usage
=====

To get a sequence code in the field configured to store sequence codes:

1. Create a new record. The sequence field should be readonly to the
   user.
2. Or open a record with empty sequence field, click the action menu,
   and click ``Set [name of field]``.
3. The record has a sequence code in the sequence field.
